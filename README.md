# Getting started

Install deps
```shell
npm install
```

## Developing

Run TSC in watch mode & restart bot via nodemon

```shell
npm run watch
```

### After making the changes:
Commit

```shell
npx git-cz
```

Prepare release

```shell
npm run release
```

Bump version

```shell
npm run bump
```

Publish release

```shell
npm run publish
```

## Usage

Start bot

```shell
npm run start
```
